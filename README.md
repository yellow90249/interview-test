# UNNOTECH 測驗 by Gary

## 執行專案

拉下來後安裝依賴

```
npm install
```

再啟動本地 server

```
npm run serve
```

輸入路徑 `/books` 到書本列表

補充 : 若起不來，嘗試把 `package-lock.json` 和 `node_modules` 都刪掉在從頭試一次。
